# 📚 Study Buddy — AI Study Material Generator

> Transform any PDF or text into summaries, flashcards, structured notes, and exam questions — instantly powered by AI.

---

## ✨ Features

| Tool | Description |
|---|---|
| 📄 **PDF Summarizer** | Upload any PDF → get 6–10 bullet-point key ideas |
| 🃏 **Flashcard Creator** | Paste text or PDF → get Q&A study cards |
| 📝 **Note Generator** | Enter a topic → notes at Low / Medium / Advanced depth |
| ❓ **Question Generator** | Any content → 5 most important exam questions |

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, Vanilla JavaScript, PDF.js
- **Backend:** Python, Flask
- **AI Model:** LLaMA 3.1 via [Groq API](https://console.groq.com) (free)
- **PDF Parsing:** pdfplumber
